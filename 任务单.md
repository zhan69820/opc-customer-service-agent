# 客服 Agent 开发任务

## 任务单
- 标题: 客服 Agent MVP 开发
- 类型: 技术开发
- 优先级: P0 🔴
- 负责人: 开发工程师 + 代码审查员
- 描述: 开发一个可商用的客服 AI Agent，支持多平台接入、自动回复、知识库问答
- 交付物: 可运行的客服 Agent 系统 + 部署文档
- 截止: 3天内

---

## 产品需求

### 核心功能
1. **自动问答**: 基于知识库回答常见问题
2. **关键词触发**: 关键词匹配触发指定回复
3. **多平台接入**: 微信、小红书、网页
4. **人机协作**: AI 无法回答时转人工

### 目标用户
- 电商卖家
- 微商
- 线下门店

### 定价
¥99/月

---

## 技术要求

- 对话模型: 接入大模型 API
- 知识库: 支持上传 QA 问答对
- 部署: 支持 Docker 一键部署
- 维护: 简单易用的管理后台
